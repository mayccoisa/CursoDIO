Tags no HTML

(voltar)

As tags são utilizadas para informar ao navagador que tipo de estrutura é essa que está sendo construída, podendo ser títulos, parágrafos, imagens, links, entre outros.

Algumas tags que aprendemos são:

  • < i > - Deixa o texto em Italico
  • < strong > - Deixa o texto em bold
  • < input > - Chama um campo do tipo Input
  • < img > - Adiciona uma imagem
  • < mark > - Cria uma marcação no fundo do texto, semelhante a um marca-texto
  • < blockquote > - Adiciona o formato de citação de texto
  • < p > - Cria um paragráfo de texto
  • < hr > - Adiciona um divider entre os elemntos acima e abaixo
  • < small > - Adiciona um texto pequeno
  • < h1,h2,h3,h4,h5,h6 > - Cria títulos de tamanhos diferentes, quanto maior número que acompanha o h, menor o tamanho do título
  • < ol > < li > - Adiciona uma lista ordenada
  • < ul > < li > - Adiciona uma lista de pontos
  • < a > - Linka páginas externas/internas (entenda páginas internas como páginas de um mesmo site) ou acesso a áreas de uma p;agina

Atributos no HTML

(voltar)

Algumas tags tem atributos atrelados a elas, os atributos são palavras especiais usadas dentro da tag de abertura para controlar o comportamento do elemento, esses atributos modificão elementos no HTML

Alguns atributos que aprendemos são:

  • color - atributo da tag input, serve altera a cor do elemento
  • number - atributo da tag input, serve para adicionar n;umericos
  • text - atributo da tag input, serve para adicionar um campo de texto
  • width - atributo da tag src, controlar a largura da imagem
  • color - atributo da tag input, serve altera a cor do elemento
